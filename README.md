# Flask Project Documentation

## Overview
This project is a web application built using Flask framework. It enables users to manage tasks effectively.

## Features
- User authentication
- Task management
- Real-time notifications

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Vishva1525/FlaskProject.git
   cd FlaskProject
   ```
2. Create a virtual environment:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   ```
3. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```
4. Set up environment variables and run the application:
   ```bash
   export FLASK_APP=app.py
   flask run
   ```

Enjoy using the project!
